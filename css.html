Introduction to CSS

What is CSS?

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in HTML or XML. CSS controls the layout of multiple web pages all at once.

How CSS Works

CSS allows you to control the color, font, size, spacing, positioning, and layout of elements on a webpage. By separating the content (HTML) from the style (CSS), you can make the page easier to maintain.

Basic Syntax of CSS

CSS is written in rules that define how elements on the page are styled. A rule consists of a selector and a declaration block.

Example CSS Rule:

Selector: h1 (targets all <h1> elements)
Declaration Block: color: blue; font-size: 24px; (defines the properties to style the selected element)

So, the complete CSS rule looks like:

h1 { color: blue; font-size: 24px; }

CSS Selectors

Selectors are used to target the HTML elements you want to style. Common selectors include:

  • h1, h2, p – Targets all <h1>, <h2>, and <p> elements.
  • #idname – Targets an element with a specific ID attribute.
  • .classname – Targets elements with a specific class attribute.
