Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure and content of a webpage, using a system of tags and attributes to organize text, images, links, and other elements.

Common HTML Tags

Here are some of the most common HTML tags used to structure a webpage:

  • <html> – The root element that contains all the other elements.
  • <head> – Contains metadata and links to stylesheets or scripts.
  • <body> – Contains the visible content of the webpage, like text and images.
  • <h1> to <h6> – Heading tags, with <h1> being the largest and <h6> the smallest.
  • <p> – Represents a paragraph of text.
  • <a> – Creates a hyperlink.
  • <img> – Used to embed images.
